# SEO Auditor Skill

When auditing SEO and discoverability:

## On-Page SEO (0-8 points)

### Meta Tags
- [ ] Title tag present and under 60 chars?
- [ ] Meta description present and under 160 chars?
- [ ] Title includes primary keyword?
- [ ] Each page has unique title/description?
- [ ] Open Graph tags for social sharing?

### Content Structure
- [ ] One H1 per page?
- [ ] Logical H2/H3 hierarchy?
- [ ] Headers include relevant keywords?
- [ ] Content length adequate for topic (1500+ words for key pages)?
- [ ] Alt text on images?

### Technical SEO (0-6 points)
- [ ] Schema markup present (Organization, LocalBusiness, Product, FAQ)?
- [ ] XML sitemap exists (/sitemap.xml)?
- [ ] Robots.txt configured properly?
- [ ] Canonical URLs set?
- [ ] SSL/HTTPS enabled?
- [ ] Mobile-responsive design?

### Internal Linking (0-6 points)
- [ ] Key pages linked from navigation?
- [ ] Blog posts link to relevant service/product pages?
- [ ] Anchor text is descriptive (not "click here")?
- [ ] No orphan pages (pages with zero internal links)?
- [ ] Breadcrumbs present?
- [ ] Footer contains important page links?

## Critical SEO Issues to Flag
- Zero meta descriptions across the site
- Duplicate title tags
- Missing H1 tags
- No schema markup
- Broken internal links
- No sitemap.xml
- Mixed content (HTTP resources on HTTPS site)

## Output
Score: X/20 for SEO & Discoverability
Each finding with severity, current state, and recommended fix
