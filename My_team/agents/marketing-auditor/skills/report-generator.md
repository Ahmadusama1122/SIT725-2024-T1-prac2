# Report Generator Skill

When generating the final audit report:

## Report Structure

```
# Marketing Audit Report

**Business:** [Name]
**Website:** [URL]
**Date:** [Date]
**Business Type:** [SaaS/Local/E-commerce/etc.]

---

## Overall Score: [X]/100 — Grade: [A/B/C/D/F]

### Grading Scale
- 90-100: A (Excellent)
- 80-89: B (Good)
- 70-79: C (Average)
- 60-69: D (Below Average)
- Below 60: F (Needs Major Work)

---

## Executive Summary
3-4 sentences: What was audited, key strengths, biggest gaps, overall assessment.

---

## Score Breakdown

| Category | Score | Grade | Status |
|----------|-------|-------|--------|
| Content & Messaging | X/20 | | |
| Conversion Optimization | X/20 | | |
| SEO & Discoverability | X/20 | | |
| Competitive Position | X/20 | | |
| Strategy & Growth | X/20 | | |
| **Total** | **X/100** | | |

---

## Key Findings

### Critical (Fix Immediately)
1. [Finding with specific details and evidence]

### High (Fix This Week)
1. [Finding]

### Medium (Fix This Month)
1. [Finding]

### Low (Nice to Have)
1. [Finding]

---

## Prioritized Action Plan

### Quick Wins (This Week)
- [ ] Action item 1
- [ ] Action item 2

### Medium-Term (1-3 Months)
- [ ] Action item 1
- [ ] Action item 2

### Strategic (3-6 Months)
- [ ] Action item 1
- [ ] Action item 2

---

## Competitive Landscape

### Key Competitors
| Competitor | Strengths | Weaknesses | Threat Level |
|-----------|-----------|------------|-------------|

### Differentiation Opportunities
- Opportunity 1
- Opportunity 2

---

## Methodology
This audit analyzed [X] pages across 5 marketing dimensions using AI-powered analysis tools. Scores are benchmarked against industry standards for [business type] businesses.
```

## Formatting Rules
- Use clean Markdown throughout
- Include specific numbers and evidence for every finding
- Make action items specific enough that someone could execute them immediately
- Sort findings by business impact, not technical severity
- Keep executive summary under 100 words
- Every recommendation should answer: What to do, Why it matters, Expected impact
